# TASK-002: 사용자·역할·프로젝트 권한 기반 구성

TASK-001 완료 후 수행합니다.

## 목적

실제 Entra 연동 전에 개발용 인증 대체수단을 통해 역할·프로젝트 범위 권한을 서버에서 검증하는 기반을 만듭니다.

## 핵심 완료 조건

- User, Department, Role, Permission, UserProjectAccess 기본 모델
- 서버 정책 기반 권한검사
- 권한 없는 프로젝트 API 접근 거부 테스트
- UI 숨김과 무관하게 API가 거부되는 통합 테스트
- 운영에서는 개발용 인증을 사용할 수 없도록 환경 차단
