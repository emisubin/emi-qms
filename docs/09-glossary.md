# 9. 용어집

| 용어 | 정의 |
|---|---|
| 프로젝트 | 고객 수주 또는 PJT 단위 |
| 제품·판넬 | 실제 생산·검사·출하 상태를 갖는 개별 추적 단위 |
| Revision | 도면, BOM, 검사양식 등의 변경 버전 |
| NCR | 부적합 보고 건. 항목별 독립 번호를 가짐 |
| 조치 | 제조 등이 부적합 상태를 수정한 기록 |
| 재검사 | 조치 이후 품질이 별도로 수행한 검사 |
| 품질 승인 | 필수 검사와 NCR 종료를 확인한 출하 허용 상태 |
| 감사 로그 | 누가 언제 무엇을 조회·변경·승인했는지 남긴 기록 |
| 정정 요청 | 승인 완료 데이터를 직접 수정하지 않고 변경하기 위한 통제 절차 |
| 스냅샷 PDF | 승인 당시 데이터와 양식 Revision을 고정한 문서 |
