# 6. MVP 범위

## 포함

1. Microsoft Entra ID 기반 로그인 준비 구조
2. 사용자·부서·역할·프로젝트 권한
3. 영업 프로젝트 등록과 면수 기준 제품·패널 자동 생성
4. 제품별 QR 발급·재발급 이력
5. 설계 패널명·W/H/D(mm) 사이즈 입력
6. 구매정보, 생산계획, 제조 10단계 작업 입력
7. 검사계획, 품질 체크리스트, 필수 사진
8. NCR, 제조 조치, 품질 재검사
9. 품질 승인 및 승인 시점 PDF
10. 포장 체크리스트, 배차, 출발, 납품 증빙
11. 제품·프로젝트 상태 대시보드
12. 내부 업무함과 핵심 이벤트 알림
13. 감사 로그와 승인 데이터 정정 절차
14. 반응형 웹과 PWA 운영 준비

## 후속 단계

- ECOUNT 실시간 연동
- 구매·생산 엑셀 고급 동기화
- 박스·포장 전용 QR
- 고객사 외부 포털
- Service Worker, 오프라인 캐시, 오프라인 입력
- AI 사진 판정
- 공정시간 예측 및 납기 위험 예측

MVP에서는 핵심 이력의 정확성, 권한, 상태 통제를 우선하며 고급 분석은 뒤로 미룹니다.

## 현재 작업에서 제외

TASK-002A는 프로젝트 등록 API, 프로젝트 DB 테이블 확장, 프로젝트 화면, Service Worker, Microsoft Entra ID 연동, QR 생성, 구매·제조·품질·알림 기능을 구현하지 않습니다.
